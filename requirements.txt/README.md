# Лабораторна робота: Структури даних Python

## Опис
Виконання завдань з імплементації та маніпуляції типами даних (list, tuple, dict, set).

## Як запустити
1. Переконайтеся, що встановлено Python 3.x.
2. Клонуйте репозиторій:
   `git clone [https://github.com/MaxymGantelya/Python]`
3. Запустіть програму:
   `python main.py`
