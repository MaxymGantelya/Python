# Лабораторна робота №2: Об'єктно-орієнтоване програмування в Python

## Опис
Імплементація принципів ООП: наслідування, інкапсуляція, поліморфізм, абстрактні класи та патерни проектування.

## Як запустити
1. Переконайтеся, що встановлено Python 3.x.
2. Клонуйте репозиторій:
   `git clone [посилання_на_мій_репозиторій]`
3. Запустіть програму:
   `python main.py`
